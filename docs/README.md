# StudyFlow – Smart Study Planning System

## Project Description
StudyFlow is a software engineering term project that aims to help students
organize exams, generate personalized study plans, and track their learning progress.

## Problem Statement
Students often struggle with managing multiple exams and creating effective
study schedules manually. This leads to inefficient learning and poor time management.

## Proposed Solution
StudyFlow provides an intelligent study planning system that automatically
generates study schedules based on exam dates, topics, and student progress.
